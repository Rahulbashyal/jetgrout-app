Installation Guide üöÄ

This guide will walk you through the complete installation process of the JetGrout application, from extracting the project files to accessing the application in your browser.

Prerequisites

Before you begin, ensure you have the following installed on your system:

  • XAMPP (includes Apache, MySQL/MariaDB, and PHP)
  • Web Browser (Chrome, Firefox, Safari, or Edge)
  • Text Editor (optional, for configuration)

Step 1: Extract the Project Files

  1. Locate the downloaded jetgrout-app.zip file on your computer
  2. Right-click on the ZIP file and select "Extract All..." or use your preferred extraction tool
  3. Navigate to your XAMPP installation directory:
    • Windows: C:\xampp\htdocs\
    • Linux: /opt/lampp/htdocs/
    • Mac: /Applications/XAMPP/htdocs/
  4. Extract or copy the jetgrout-app folder into the htdocs directory
  5. Verify that the folder structure looks like this: 
    htdocs/

‚úì Verification

After extraction, you should see the jetgrout-app folder containing all the Laravel application files including app, public, resources, and other directories.

Database Setup üóÑÔ∏è

The JetGrout application requires a MySQL database to store all project data, user information, and daily reports.

Step 1: Start MySQL Service

  1. Open the XAMPP Control Panel
  2. Click the "Start" button next to Apache
  3. Click the "Start" button next to MySQL
  4. Wait for both services to show green "Running" status

‚ö†Ô∏è Port Conflicts

If Apache or MySQL fails to start, it may be due to port conflicts. Common solutions:

  • Close Skype or other applications using port 80 or 443
  • Stop other MySQL/MariaDB services running on your system
  • Configure XAMPP to use different ports in the Config menu

Step 2: Access phpMyAdmin

  1. Open your web browser
  2. Navigate to http://localhost/phpmyadmin
  3. You should see the phpMyAdmin interface

Step 3: Create the Database

  1. In phpMyAdmin, click on the "Databases" tab at the top
  2. In the "Create database" section, enter the database name: jetgrout_db
  3. Select "utf8mb4_unicode_ci" as the collation
  4. Click the "Create" button

Step 4: Import the Database Structure

There are two methods to set up your database tables:

Method 1: Import SQL File (Recommended)

  1. In phpMyAdmin, click on the "jetgrout_db" database you just created
  2. Click on the "Import" tab
  3. Click "Choose File" and navigate to: jetgrout-app/database/jetgrout_db.sql
  4. Click "Go" at the bottom of the page
  5. Wait for the import to complete - you should see a success message

Method 2: Run Laravel Migrations

  1. Open Terminal (Linux/Mac) or Command Prompt (Windows)
  2. Navigate to the project directory: 
    cd /opt/lampp/htdocs/jetgrout-app
  3. Run the migration command: 
    php artisan migrate
  4. Wait for all migrations to complete

‚úì Verification

After importing, you should see multiple tables in the jetgrout_db database including:

  • users - User accounts
  • projects - Project information
  • daily_reports - Daily site reports
  • drilling_logs - Drilling data
  • grouting_logs - Grouting data
  • And more...

Step 5: Configure Database Connection

  1. Navigate to the project folder: jetgrout-app
  2. Find and open the .env file with a text editor
  3. Locate the database configuration section and update it: 
    DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=jetgrout_db
DB_USERNAME=root
DB_PASSWORD=
  4. Save the file

Database Configuration Explained

  • DB_CONNECTION: Database type (mysql)
  • DB_HOST: Database server address (127.0.0.1 for local)
  • DB_PORT: MySQL port (default: 3306)
  • DB_DATABASE: Database name (jetgrout_db)
  • DB_USERNAME: Database username (default: root)
  • DB_PASSWORD: Database password (leave empty for XAMPP default)

What is JetGrout?

JetGrout is a digital construction management system designed specifically for jet grouting operations. It replaces traditional paper-based logging with a modern, cloud-based solution that enables real-time data collection, photo documentation, and automated reporting.

Key Features

  • Real-time Data Entry: Field workers can enter data directly from the construction site
  • Photo Documentation: Capture and attach photos of soil samples and pressure gauges
  • Role-based Access: Different permissions for Admin, Supervisor, Driller, and Grouter roles
  • Automated Reports: Generate comprehensive daily reports automatically
  • Mobile Friendly: Works on smartphones, tablets, and desktop computers
  • Cloud Storage: All data safely stored and accessible from anywhere

Login Process

To begin using JetGrout, you need to log in with your provided credentials.

Steps to Login

Upon successful login, you will be automatically redirected to your role-specific dashboard where you can begin managing your construction data.

Frequently Asked Questions

Q: I forgot my password. What do I do?

A: Contact your administrator. They can reset your password for you. DO NOT create a new account as you will lose access to your old data.

Q: The camera button doesn't work on my phone. Why?

A: Your browser needs permission to use the camera.

On Android:

  1. Go to browser Settings
  2. Find "Site Settings" or "Permissions"
  3. Find the JetGrout website
  4. Allow "Camera" permission

On iPhone:

  1. Go to iPhone Settings
  2. Scroll to your browser (Chrome/Safari)
  3. Turn ON "Camera" permission

Alternative: Use the "Upload" button and select photos from your gallery.

Q: I entered wrong data and already saved. Can I fix it?

A: YES! You can edit your data.

For Drillers:

  1. Click "View My Logs"
  2. Find the wrong entry
  3. Click "Edit"
  4. Fix the mistake
  5. Click "Update"

For Grouters:

  1. Click "View My Submissions"
  2. Find the wrong entry
  3. Click "Edit"
  4. Fix the mistake
  5. Click "Update All Data"

Q: Can I use this on my phone?

A: YES! JetGrout works on:

  • ‚úì Desktop computers
  • ‚úì Laptops
  • ‚úì Tablets (iPad, Android tablets)
  • ‚úì Smartphones (iPhone, Android phones)

Just open your web browser and navigate to the JetGrout URL.

Q: What if there's no internet at the site?

A: You need internet to use JetGrout. Options:

  1. Use mobile data (4G/5G) on your phone
  2. Set up WiFi at the site office
  3. Record data on paper temporarily, then enter it when you have internet

Note: Future updates may include offline mode functionality.

Q: I can't see the data my colleague entered. Why?

A: Check these things:

  1. Are you looking at the correct project?
  2. Are you looking at the correct date?
  3. Did your colleague actually SAVE their data?
  4. Try refreshing the page (press F5)

If still not visible, ask your admin to check permissions.
